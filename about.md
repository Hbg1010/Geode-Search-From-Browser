# Search from Browser

This mod allows the user to search in gd from their browser!


# How to use
In your browser, you can use the following:

Geode://level/<Name/ID> 

Geode://list/<Name/ID> 

Geode://user/<Username/ID>

Geode://mappacks <cb>:terror:</c>

The search menu will then open up in game!
